# Physical Security: Cameras and Locks

Physical security is a critical layer in network security. It protects networking equipment, servers, and other sensitive infrastructure from unauthorized access or damage. Two common methods of physical security are cameras and locks.

---

## 1. Cameras

Cameras are used for monitoring and deterrence.

- **Purpose:**
  - Monitor entrances, exits, server rooms, and critical areas.
  - Deter unauthorized access or theft.
  - Provide evidence in case of security incidents.

- **Types of Cameras:**
  - **IP Cameras:** Connect directly to the network, allow remote monitoring, and can be integrated with video management systems.
  - **Analog Cameras:** Traditional CCTV cameras connected to a recording device.
  - **PTZ Cameras (Pan-Tilt-Zoom):** Can rotate and zoom in to cover larger areas or focus on specific events.

- **Best Practices:**
  - Position cameras at all entry points and sensitive areas.
  - Ensure cameras are tamper-resistant.
  - Use proper lighting for clear footage.
  - Implement access control for video footage storage.

## 2. Locks

Locks physically restrict access to sensitive areas and devices.

- **Purpose:**
  - Prevent unauthorized physical access to network devices, server racks, and rooms.
  - Protect equipment from tampering or theft.

- **Types of Locks:**
  - **Traditional Key Locks:** Standard mechanical locks for doors or cabinets.
  - **Electronic Locks / Keypads:** Require a code, card, or key fob to unlock.
  - **Biometric Locks:** Use fingerprints, retinal scans, or other biometric data for access.
  - **Cable Locks:** Physically secure laptops, desktops, or small network devices.

- **Best Practices:**
  - Limit key or access code distribution to authorized personnel only.
  - Regularly change codes and reissue keys as needed.
  - Combine locks with surveillance for maximum security.
  - Lock network racks and server rooms at all times.

## Summary

Physical security complements logical security measures. Cameras provide monitoring and evidence, while locks restrict access. Both are essential to protect hardware and sensitive network data from physical threats.
